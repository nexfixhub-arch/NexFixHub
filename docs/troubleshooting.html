Troubleshooting

Common Issues

Authentication Error

Problem: "Invalid API Key" error

Solution:

  1. Verify your API key is correct
  2. Check that the key is not expired
  3. Ensure the key has necessary permissions
  4. Regenerate the key if needed

Repository Not Found

Problem: "Repository not found" error

Solution:

  1. Verify the repository path is correct (owner/repo)
  2. Ensure the repository is public or you have access
  3. Check your API key permissions

Timeout Issues

Problem: "Request timeout" error

Solution:

  1. Increase the timeout value in configuration
  2. Check your internet connection
  3. Try with a smaller repository first
  4. Use 'quick' analysis depth instead of 'full'
