API Reference

analyzeRepository()

Analyzes a GitHub repository for issues and vulnerabilities.

hub.analyzeRepository(repositoryPath, options)

Parameters

  • repositoryPath (string) - Repository in format "owner/repo"
  • options (object) - Optional configuration

Returns

{
    issues: Array,
    vulnerabilities: Array,
    suggestions: Array,
    score: Number
}

getIssueDetails()

Get detailed information about a specific issue.

hub.getIssueDetails(issueId)

suggestFix()

Get AI-powered suggestions for fixing an issue.

hub.suggestFix(issueId)

Error Handling

All API calls return errors in the following format:

{
    error: true,
    code: 'ERROR_CODE',
    message: 'Error description'
}
