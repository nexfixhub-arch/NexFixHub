Configuration

Configuration Options

NexFixHub can be configured with various options:

const hub = new NexFixHub({
    apiKey: 'your-api-key',
    timeout: 30000,
    retries: 3,
    logLevel: 'info',
    analysisDepth: 'full',
    excludePatterns: ['node_modules', 'dist']
});

Environment Variables

  • NEXFIXHUB_API_KEY - Your API key
  • NEXFIXHUB_TIMEOUT - Request timeout in milliseconds
  • NEXFIXHUB_LOG_LEVEL - Log level (debug, info, warn, error)

Analysis Depth

Choose the depth of analysis:

  • quick - Fast surface-level analysis
  • standard - Balanced analysis (default)
  • full - Deep comprehensive analysis

Exclude Patterns

Configure which files and directories to exclude from analysis:

excludePatterns: [
    'node_modules',
    'dist',
    'build',
    '*.min.js',
    '.git'
]
