Getting Started

Installation

To get started with NexFixHub, follow these simple steps:

npm install nexfixhub

Or using yarn:

yarn add nexfixhub

Quick Start

Create your first analysis in just a few minutes:

const NexFixHub = require('nexfixhub');

const hub = new NexFixHub({
    apiKey: 'your-api-key'
});

// Analyze your repository
await hub.analyzeRepository('owner/repo');

Authentication

To authenticate with NexFixHub, you need an API key:

  1. Create an account on NexFixHub
  2. Generate an API key from your dashboard
  3. Set it in your environment: NEXFIXHUB_API_KEY=your-key
